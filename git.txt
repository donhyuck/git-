12. 다른 공간B에서 기존작업이어하기(pull)
<다른 작업공간 내>
1) 다른 작업공간에서도 일단 저장소는 만들어야 받을 수 있다.
> git init

2) 저장소--(연결)--다른 작업공간
> git remote origin ~~작업git주소~~

3) pull로 기존작업가져오기
> git pull origin master
